RDC Network

O que é?

RDC Network é um projeto de comunicação via bluetooth descentralizado

Detalhes

  • Possui comunicação offline (via bluetooth)
  • É descentralizado
  • É grátis
  • É rápido e seguro (escrito em Rust)

    • *Possui apenas mensagens de texto (pode ser um contra para quem está acostumado com redes sociais mais famosas)
